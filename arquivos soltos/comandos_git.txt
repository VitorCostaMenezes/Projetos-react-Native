branch

são versões diferentes do seu sistema;


após criar um arquivo, vc deve nserir o comando:

git status	  -	 para vizualizar os aruivos na pasta


para adicionar os arquivos não traqueados vc utiliza:


git add   arquivo_a_ser_usado

ou

git add -A       -  para adicionar todos os arquivos.




realizar um commit:

git commit -m "frase desejada para identificar as alterações realizadas"




realizar um commit e ja adicionar os arquivos não trackeados:

git commit -am "frase desejada para identificar as alterações realizadas"







Vizualizar todos os commits feitos o branch:

git log
